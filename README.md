<h2  align="center">Cookie-Clicker_Clicker</h2>
<p  align="center">A automatic clicker for Cookie Clicker http://orteil.dashnet.org/cookieclicker/</p>

<br/>

## Installing

1. Clone this repo ```git clone https://github.com/tokyojack/Cookie-Clicker_Clicker```
2. Run ```pip install mouse selenium APScheduler``` for the packages
3. Run it with ```python app.py```


## Usage

The application will automatically start clicking for you when it opens. Just continue playing normally and it'll click the cookie.
